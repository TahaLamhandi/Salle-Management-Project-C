✔️ Salle Management System ✔️

Welcome to the Salle Management System! This is a console-based application developed in C to efficiently manage reservations for venues such as Salles TD, Salles TP, and Amphis. Whether you're a student or an administrative agent, this system ensures smooth scheduling and resource allocation for events.

🌟 Features :

1. Student Space

   Students can reserve venues for nighttime sessions.

   Perfect for organizing evening events or study sessions.

2. Administration Space

    Accessible only with a unique code provided by an administrative agent.

    Allows reservation of venues for morning sessions.

    Supports cancellation of reservations.

3. Venue Types

    Salles TD: Ideal for small group discussions.

    Salles TP: Perfect for practical sessions.

    Amphis: Large venues for lectures and events.

4. Role-Based Access

    Students: Limited to nighttime reservations.

    Administrative Agents: Full control over morning sessions and cancellations.
